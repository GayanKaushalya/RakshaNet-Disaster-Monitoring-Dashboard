# 🌍 RakshaNet Disaster Monitoring Dashboard

### Frontend for the Hyper-Local Landslide & Flood Prediction System

The **RakshaNet Frontend** is a modern web dashboard built to visualize environmental sensor data and disaster risk predictions generated by the RakshaNet AI backend.

It provides a **real-time monitoring interface** for tracking environmental conditions and receiving early warnings for **landslides and floods**.

This dashboard communicates with the **RakshaNet backend prediction engine** to display live sensor readings, prediction results, and disaster alerts.

# 🧠 Features

- 📊 Real-time environmental monitoring dashboard  
- 🌧 Rainfall, soil moisture, and water level visualization  
- 🚨 Disaster risk alerts from the AI prediction system  
- 📡 Live data updates from IoT sensor simulations  
- 🖥 Clean and responsive UI dashboard  
- 🔗 Integration with RakshaNet backend prediction API

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/rakshanet-frontend.git
cd rakshanet-frontend
```

Install dependencies

```bash
npm install
```

---

# ▶ Running the Development Server

```bash
npm run dev
```

The application will start on:

```
http://localhost:5173
```
